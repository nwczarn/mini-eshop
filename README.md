# MiniEshop 🛒

Microservices-based e-commerce demo built with .NET and Docker.

## 🚀 Tech Stack
- .NET 8
- Docker & Docker Compose
- Microservices architecture
- (coming soon) Kafka, RabbitMQ, Redis

## 🧩 Services
- Storefront (8080)
- Admin (8081)

## ▶️ Run locally
```bash
docker-compose -f docker-compose.override.yml up --build
